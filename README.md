# Actividades-DMOJ-Puig-Castellar
Aqui presto todas las actividades hecha del DMOJ del instituto Puig Castellar

Cabe recalcar que este repositorio es para ayudar a aquellas personas que se rindieron intentando resolver una actividad de este DMOJ y puedan relajarse copiando o pegando, o como guia para resolverlo por vuestra propia cuenta, o para aquellos obsesionado con completar cosas al 100% incluso para esto les sea más fácil XD.

Otra cosa a destacar es que algunas de estas actividad fue hecho en base a ayuda externa (IA), por ende, aquellas actividades que parezcan muy complejas / professionales son hechas con dicha ayuda.

Disfruten de un buen descanso mental con este repositorio.
