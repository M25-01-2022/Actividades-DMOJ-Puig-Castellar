# Actividades-DMOJ-Puig-Castellar
Aquí presto todas las actividades hechas del DMOJ del instituto Puig Castellar

Cabe recalcar que este repositorio es para ayudar a aquellas personas que se rindieron intentando resolver una actividad de este DMOJ y puedan relajarse copiando o pegando, o como guía para resolverlo por vuestra propia cuenta, o para aquel obsesionado con completar cosas al 100% incluso para esto les sea más fácil XD.

Otra cosa a destacar es que algunas de estas actividades fue hecho con ayuda externa (IA), por lo tanto, aquellas actividades que parezcan muy complejas o profesionales son hechas con dicha ayuda.

Para poder encontrar la actividad que desees fíjate en el nombre de la actividad y ponlo en el buscador
